## 第一步使用yum安装Nginx

```
yum install nginx	
```

这一步非常简单，Linux本身就具备了非常优秀的包管理

## 启动Nginx

启动

```
nginx 
netstat -anp | grep 80 
```

第一行用于启动Nginx

第二行用于查看80端口的占用情况

此时就可以去公网ip查看Nginx的欢迎页了

## Nginx的语法

```
man nginx 
```

可以使用linux自带的手册查看：

其中常用的就是向nginx传递signal

```
nginx -s signal
```

- `stop` — fast shutdown
- `quit` — graceful shutdown
- `reload` — reloading the configuration file #常用于跟新configure文件以后
- `reopen` — reopening the log files

## 配置文件的结构

配置文件包括

directives  -----> simple directives: name parameter;

​				------->block directives: { }

意思就是说包括简单指令和块指令。

http { } 这就是一个块指令

## 服务静态内容

作为一个web服务器，一个非常重要的事情就是serving out images or HTML 页面

以此为例子：我们需要在http块中写入server块。

第一步：

在 ~ 目录下创建 /data/www 文件夹 和 /data/images 文件夹

第二步：

打开位于 config文件，进行编辑

```
# For more information on configuration, see:
#   * Official English Documentation: http://nginx.org/en/docs/
#   * Official Russian Documentation: http://nginx.org/ru/docs/

user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log;
pid /run/nginx.pid;

# Load dynamic modules. See /usr/share/doc/nginx/README.dynamic.
include /usr/share/nginx/modules/*.conf;

events {
    worker_connections 1024;
}

http {
    log_format  main  '$remote_addr - $remote_user [$time_local] "$request" '
                      '$status $body_bytes_sent "$http_referer" '
                      '"$http_user_agent" "$http_x_forwarded_for"';

    access_log  /var/log/nginx/access.log  main;

    sendfile            on;
    tcp_nopush          on;
    tcp_nodelay         on;
    keepalive_timeout   65;
    types_hash_max_size 2048;

    include             /etc/nginx/mime.types;
    default_type        application/octet-stream;

    # Load modular configuration files from the /etc/nginx/conf.d directory.
    # See http://nginx.org/en/docs/ngx_core_module.html#include
    # for more information.
    include /etc/nginx/conf.d/*.conf;

    server {
        listen       80 default_server;
        listen       [::]:80 default_server;
        server_name  _;
        root         /usr/share/nginx/html;

        # Load configuration files for the default server block.
        include /etc/nginx/default.d/*.conf;

        location / {
        }

        error_page 404 /404.html;
        location = /404.html {
        }

        error_page 500 502 503 504 /50x.html;
        location = /50x.html {
        }
    }
```
