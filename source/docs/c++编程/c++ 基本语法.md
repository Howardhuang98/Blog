## 变量

变量存在的意义，方便我们管理我们的内存空间。

创建变量方法: `数据类型 变量名 = 初始值;`

```
int a = 10;
```

## 常量

常量记录程序中不可更改的数据，它有两种定义方法：

1. `#define A 7`
2. `const int A = 7`

## 数据类型

数据类型是为了明确给变量分配合适的内存空间。

2位表示，第一位正负，-2，-1，0，1--->取值范围  2^(n-1),2^(n-1)-1

| 数据类型  | 大小     | 取值范围        |
| --------- | -------- | --------------- |
| short     | 2B=16bit | -2^15,2^15-1    |
| int       | 4B       |                 |
| long      | 4B       |                 |
| long long | 8B       |                 |
| float     | 4B       | 7位有效数字     |
| double    | 8B       | 15-16位有效数字 |
| char      | 1B       | 存一个ascii码   |
