闭包

def out_func(a,b):
    c = 10
    def inner_func():
        s = a + b + c
        print(s)
    return inner_func

#调用out
inner = out_func(1,2)
inner()

以上是闭包的基本格式，可以看到return是可以返回变量，函数任何内部的对象
