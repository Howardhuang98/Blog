双向队列

collections.deque

Deque队列是由栈或者queue队列生成的（发音是 “deck”，”double-ended queue”的简称）。Deque 支持线程安全，内存高效添加(append)和弹出(pop)，从两端都可以，两个方向的大概开销都是 O(1) 复杂度。

list如果从前面弹出，需要耗费O(n)时间，不建议当成队列使用。
